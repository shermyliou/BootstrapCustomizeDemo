# Bootstrap 客製化演示專案 (BootstrapCustomizeDemo)

本專案是一個基於 **Vue 3** 和 **Vite** 的開發演示，旨在展示如何深度客製化 **Bootstrap 5** 的樣式系統。

## 主要功能
- **Bootstrap 變數覆蓋**：展示如何在 SCSS 中自訂顏色、間距、圓角及陰影。
- **語義化開發**：利用 Bootstrap 的 SCSS 架構進行樣式擴充。
- **樣式驗證組件**：內建 `BootstrapChecker` 組件，用於即時檢視客製化樣式的生效情況。

## 技術堆疊
- Framework: Vue 3 (Composition API)
- Build Tool: Vite
- Styling: Sass (SCSS) + Bootstrap 5
